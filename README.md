# Leitor de Arquivos OFF — Trabalho da Disciplina de Computação Gráfica

Este projeto foi desenvolvido como parte da disciplina de **Computação Gráfica**, com o objetivo de implementar uma estrutura de dados capaz de **ler, armazenar e exibir malhas geométricas** definidas no formato **OFF (Object File Format)**.

O trabalho consiste em:

- Implementar uma estrutura do tipo **Face–Vértice**
- Ler arquivos OFF contendo vértices e faces
- Armazenar dinamicamente os dados da malha
- Exibir as informações lidas (vértices e faces)
- Organizar o código em arquivos `.c` e `.h`
- Tratar malhas **triangulares** e **mistas** (polígonos com número variável de vértices)

A proposta é treinar o uso de estruturas de dados para representar modelos geométricos e manipular arquivos utilizados em Computação Gráfica.
